# Autodesk CFD 結果の取得手順（数値データ・画像）

解析が完走したあと，各ケース（**Case A / B / C** × **層流 / SST k-ω**）について
**推力・トルクの数値**と**流れ場の画像**を取り出し，レポートに載せるまでの手順．
設定・実行は [11_autodesk_cfd_gui.md](11_autodesk_cfd_gui.md)，授業資料は `Exercise05.pdf` §3 を参照．

> 取るべきもの（1 ケースあたり）：① **\(|F_z|\)（層流・SST）** と \(M_z\)，② 推力の**時系列グラフ**，
> ③ **圧力コンタ画像**（断面），④ **等圧面画像**，⑤（任意）**アニメーション avi**．

---

## 0. 出力ファイルの場所

解析を実行すると，アセンブリ（または取り込んだ STEP）と同じ場所に CFD 作業フォルダができる：

```
…\<スタディ名>\設計 1\シナリオ 1\
    シナリオ 1_******.csv   ← 数値データ（時系列の力・トルク）
    （メッシュ・解・アニメーション等）
```

- 数値は **CSV**（§1），画像は **結果タブから書き出す**（§2–§3）．
- ケース／乱流モデルごとにスタディ（またはシナリオ）を分けておくと取り違えない．

## 1. 数値データ（推力 \(|F_z|\)・トルク \(M_z\)）— CSV から

> 壁面計算機は使えない（授業資料明記）．力学情報は CSV に時系列で保存される．

### 1.1 CSV を開く

`シナリオ 1_******.csv` を Excel／表計算で開く．列はおおむね：

```
実行時間, …, 回転軸まわり流体トルク, 流体力X, 流体力Y, 流体力Z, 流体トルクX, 流体トルクY, 流体トルクZ
```

（環境により列名・順序が違うことがある．ヘッダで **Hydraulic ForceZ / TorqueZ** に当たる列を確認する．）

### 1.2 最終 1 回転で平均する

過渡を除くため**最終 1 回転分**だけを平均する（dt 0.003・400 ステップ＝2 回転なら **0.6〜1.2 s**）．

- **推力**：`Hydraulic ForceZ`（dyne）を該当区間で平均 → **\(\times10^{-5}\)** で \(|F_z|\) ［N］
  （\(1\,\mathrm{dyne}=10^{-5}\,\mathrm{N}\)）．
- **トルク**：`Hydraulic TorqueZ`（dyne-cm）を同区間で平均 → **\(\times10^{-7}\)** で \(M_z\) ［N·m］
  （\(1\,\mathrm{dyne\text{-}cm}=10^{-7}\,\mathrm{N\cdot m}\)）．

Excel 例（時間が A 列・ForceZ が F 列の場合）：

```
=AVERAGEIFS(F:F, A:A, ">=0.6", A:A, "<=1.2") * 1e-5
```

### 1.3 推力係数

\[
C_T=\frac{|F_z|}{\rho\,n^2 D^4},\quad \rho=1.225,\ n=\tfrac{100}{60}=1.667\ \mathrm{rev/s}
\]

各ケースの掃引直径 \(D\)（[11_autodesk_cfd_gui.md](11_autodesk_cfd_gui.md) §0）：A `0.096`／B `0.0908`／C `0.099` m．

### 1.4 結果表（このテンプレを埋める）

| ケース | \(|F_z|\) 層流 [N] | \(|F_z|\) SST [N] | \(M_z\) [N·m] | \(C_T\) |
|---|---|---|---|---|
| Case A |  |  |  |  |
| Case B |  |  |  |  |
| Case C |  |  |  |  |

### 1.5 推力の時系列グラフ（収束確認）

CSV の `実行時間` を横軸，`Hydraulic ForceZ` を縦軸に**散布図／折れ線**を作る．
- 初期過渡 → 平坦（または周期振動）に落ち着くか，平均区間（0.6〜1.2 s）が妥当かを確認．
- このグラフ自体もレポートの図に使える（`assets/caseX_cfd_fz_history.png` 等で保存）．

## 2. 画像（流れ場の可視化）— 結果タブ

リボンを **「結果」** タブに切り替えてから操作する（授業資料 §3）．

### 2.1 圧力コンタ（断面）

1.「**平面**」→「**追加**」→「**編集**」．
2. 結果＝**静圧**，**法線ベクトル `(0,1,0)`**（y 断面），位置（移動スライダ）を回転中心付近へ．
3. 表示＝**シェーディング**．左下の凡例（静圧）レンジを例えば **−200〜200 Pa** に固定すると
   3 ケースで色基準が揃い比較しやすい．
4. ブレード下面が正圧・上面が負圧 → **揚力発生**が確認できる．

### 2.2 等圧面（3D）

1.「**等値面**」→「**追加**」→「**編集**」：量＝静圧，表示色＝静圧，値＝**−50 Pa**，**半透明表示**．
2. もう一つ「追加」して **+50 Pa**．→ 翼の下に正圧面・上に負圧面が立体的に見える．
3. プロペラは右クリック「**色の表示**」で単色にすると面が見やすい．

### 2.3 表示の整え

- プロペラ／外箱：右クリック「**半透明表示**」「**シェーディング**」で切替．
- **ローカル座標軸は非表示**（表示タブ）にしておく（邪魔なので）．

### 2.4 アニメーション（任意・avi）

1. 圧力レンジを固定（例 −200〜200 Pa）．
2.「**アニメーション**」→ ウインドウで時刻を全選択 → 再生（三角ボタン）で 1〜2 回転を確認．
3.「**avi を保存**」で動画出力（**フレーム間隔を 0 以外**にしないと保存できない）．

## 3. 画像の保存とレポートへの取り込み

### 3.1 画像として書き出す

- Autodesk CFD のリボン「結果」→ **「イメージ」**（画面を画像ファイルに保存）を使うか，
  無ければ OS のスクリーンショット（Windows は `Win+Shift+S`）で範囲を保存する．
- 凡例（カラーバー）が入るように画面を整えてから保存する．

### 3.2 命名規約（`assets/` に保存）

形状図（`caseX_iso.png` 等）と区別するため，CFD 結果は接頭辞 `cfd_` を付ける：

| 内容 | ファイル名 |
|---|---|
| 圧力コンタ断面 | `assets/caseX_cfd_pressure.png` |
| 等圧面 | `assets/caseX_cfd_isosurface.png` |
| 推力の時系列 | `assets/caseX_cfd_fz_history.png` |
| 流速コンタ（任意） | `assets/caseX_cfd_velocity.png` |

（`X` は A/B/C．層流と SST を分けるなら `caseX_cfd_pressure_lam.png` / `_sst.png` のように）．

### 3.3 レポート（.tex）への取り込み

```latex
\begin{figure}[htbp]
  \centering
  \includegraphics[width=0.6\textwidth]{assets/caseA_cfd_pressure.png}
  \caption{Case A の静圧コンタ（y 断面）．翼下面が正圧，上面が負圧で揚力が発生している．}
\end{figure}
```

> platex + dvipdfmx で PNG を使う場合，初回のみ `extractbb assets/*.png` でバウンディングボックスを
> 生成しておく（図が出ない／エラーになるとき）．
